# Pong Game

A soccer-themed Pong game built with Python and Pygame. Play against an AI opponent or a second player.

## How to Run

1. Install Pygame:
```bash
pip install pygame
```
2. Run the game:
```bash
python main.py
```

## Controls

**Player vs AI:** LEFT / RIGHT arrow keys

**Player vs Player:** Right paddle = UP / DOWN, Left paddle = W / S

**Other:** P = Pause, M = Mute, R = Rematch, Esc = Menu/Quit
